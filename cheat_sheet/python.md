# Python Cheat Sheet:

### Variable assignment
```python
x = 10
name = "John"
is_student = True
```

### Common data types
```python
number = 42    # int
pi = 3.14      # float
text = "Hello" # str
is_true = True  # bool
```

### Lists and dictionaries
```python
my_list = [1, 2, 3]
my_dict = {"key": "value", "age": 25}
```

### If statement
```python
if x > 5:
    print("x is greater than 5")
elif x == 5:
    print("x is equal to 5")
else:
    print("x is less than 5")
```

### For loop
```python
for item in my_list:
    print(item)
```

### While loop
```python
counter = 0
while counter < 5:
    print(counter)
    counter += 1
```

### Function definition
```python
def greet(name):
    return f"Hello, {name}!"
```

### Function call
```python
result = greet("Alice")
print(result)
```
